# useQueue

A react state hook implements simple first-in first-out (FIFO) queue.

## Reference

```ts
export function useQueue<T[]>(initialValue: T[] =[]): {
  add: (item: T) => void;
  remove: () => T;
  first: T;
  size: number;
}
```

#### Importing

<ImportPath />

#### Arguments

- **initialValue** _`T[]`_ - The initial value of the queue state. Defaults to an empty array.

#### Return

- **first** _`T`_ - The value of the first item in the queue.
- **last** _`T`_ - The value of the last item in the queue.
- **add** _`(item: T) => void`_ - Adds a value to the end of the queue.
- **remove** _`() => T`_ - Removed and returns the value from the end of the queue
