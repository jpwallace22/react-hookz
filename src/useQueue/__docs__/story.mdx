# useQueue

A state hook in the form of a first-in first-out (FIFO) queue.

  <Story story={Example} inline />

## Reference

```ts
export function useQueue<T[]>(initialValue: T[] =[]): {
  add: (item: T) => void;
  remove: () => T;
  first: T;
  size: number;
}
```

#### Importing

<ImportPath />

#### Arguments

- **initialValue** _`T[]`_ - The initial value. Defaults to an empty array.

#### Return

- **first** _`T`_ - The first item in the queue.
- **last** _`T`_ - The last item in the queue.
- **add** _`(item: T) => void`_ - Adds an item to the end of the queue.
- **remove** _`() => T`_ - Removes and returns the head of the queue.
- **size** _`number`_ - The current size of the queue.
